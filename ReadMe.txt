💻 Personal Web Page Design

🌟 Introduction
The main purpose of this project is creating a personal web page to familiariasing with HTML, CSS and OpenLayers.

🛠️ Technologies Used
The following core technologies and libraries were utilized in the development of this website:
* Visual Studio Code
* HTML: The fundamental structure of the project.
* CSS: Visual styling and layout.

🌟 Project Features

* The site has 3 different HTML files.(index,hobbies,map)
* A table with images is about my hobbies in 'hobbies' page.
* An OpenLayer map is featured in 'My Location' page, showing the Ankara coordinates.
